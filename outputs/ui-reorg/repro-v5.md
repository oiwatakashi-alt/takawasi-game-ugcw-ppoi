# UI再編ウェーブ4 再現手順

1. `npm run dev -- --host 127.0.0.1`を起動し、1280pxのlocal desktopで開く。
2. Theaterから「幕舎で準備する」→「出撃配置へ」→「選抜部隊で戦闘開始」へ進む。
3. Battleで戦闘を撤退し、「戦果報告へ」を押す。
4. After Actionの戦果要約直後に「結果を反映して幕舎へ」が見え、損耗一覧の最下部まで移動せず次ターンへ進めることを確認する。
5. 共通ヘッダーに「直前報告」ラベルがあり、長文が2行に要約され、要素のtitleで全文を参照できることを確認する。
6. 次ターンCampで資源・兵力・部隊史の持越しを確認し、「戦略マップへ戻る」でTheaterへ戻る。
7. `qa-report-v5.json`と`20`〜`22`のスクリーンショットを照合する。

戦略mapと戦術mapを分離し、主戦場を省略せず、小任務だけauto-resolve可能という既存境界を維持する。
