# UI再編ウェーブ2 再現手順

1. `npm run dev -- --host 127.0.0.1`を起動し、1280pxのlocal desktopで開く。
2. Theater→Camp→Deploymentへ進み、「選抜部隊で戦闘開始」を押す。
3. Battleで、上部の警報レールがまとまり、直下の戦術mapが初期viewportへ出ることを確認する。
4. 3倍速で戦闘を進め、戦線崩壊後に「戦果報告へ」を押す。
5. After Actionで、左の戦術目標・戦利品・参謀責任の文字が読め、右の旅団損耗カードが部隊名と詳細に分かれていることを確認する。
6. 「結果を反映して幕舎へ」を押し、第6戦略ターンCampで資源・兵力・履歴の持越しを確認する。
7. `qa-report-v3.json`と`13`〜`15`のスクリーンショットを照合する。

Battleの主戦場mapは省略せず、戦略mapと戦術mapを分離し、小作戦だけauto-resolve可能という既存境界を維持する。
