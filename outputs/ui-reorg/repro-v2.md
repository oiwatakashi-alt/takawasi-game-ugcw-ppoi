# UI再編 v1 グラフィック適用後の再現手順

1. `npm run dev -- --host 127.0.0.1`でlocal desktopを起動する。
2. `http://127.0.0.1:5173/`を1280px幅で開き、Theater左側の5層戦線パネルに地形map textureが低コントラストで見えることを確認する。
3. 「幕舎で準備する」→「出撃配置へ」→「選抜部隊で戦闘開始」を押す。
4. Battleで戦術map、前線、部隊token、objectiveが従来どおり表示されることを確認する。
5. 3倍速で戦闘を進め、戦線崩壊後に「戦果報告へ」を押す。
6. After Actionで損耗・戦果・教訓を確認し、「結果を反映して幕舎へ」を押す。
7. 次ターンCampで資源・兵力・履歴の持越しを確認する。
8. `outputs/ui-reorg/qa-report-v2.json`のruntimeQAと、`07`〜`12`のスクリーンショットを照合する。

画像導入後も、戦略mapと戦術mapは分離し、主戦場は省略せず、小作戦だけauto-resolve可能という既存境界を維持している。
