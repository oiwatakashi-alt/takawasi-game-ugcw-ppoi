# UI再編ウェーブ3 再現手順

1. `npm run dev -- --host 127.0.0.1`を起動し、1280pxのlocal desktopで開く。
2. Theaterから「幕舎で準備する」→「出撃配置へ」へ進む。
3. Deployment左ペイン上端に「幕舎へ戻る」「選抜部隊で戦闘開始」が見え、長い敵情・敵波タイムラインを先にスクロールしなくても主要操作へ到達できることを確認する。
4. 「選抜部隊で戦闘開始」を押し、Battle上部の警報レール6枚が1280px viewport内に収まることを確認する。警報レールの直下に戦術mapが表示されることも確認する。
5. 「撤退実行」→「戦果報告へ」を押し、After Actionで戦術目標・戦利品・参謀責任・部隊別損耗が読めることを確認する。
6. 「結果を反映して幕舎へ」を押し、次ターンCampで資源・兵力・部隊史の持越しを確認する。
7. `qa-report-v4.json`と`16`〜`19`のスクリーンショットを照合する。

Battleの主戦場mapは省略せず、戦略mapと戦術mapを分離し、小作戦だけauto-resolve可能という既存境界を維持する。
