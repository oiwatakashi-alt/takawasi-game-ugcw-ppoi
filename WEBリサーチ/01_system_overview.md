# Ultimate General: Civil War システム全体像

調査日: 2026-06-30

## ざっくり構造

`Ultimate General: Civil War` は、戦闘RTSとキャンペーン育成を往復するゲーム。

基本ループは次の通り。

1. キャンペーン開始時に将軍の初期能力と陣営を選ぶ。
2. 戦闘に勝つ、引き分ける、負ける。
3. 結果に応じてゴールド、兵員、評判、キャリアポイントを得る、または失う。
4. Campで軍を再編成する。
5. 部隊を補充し、武器を買い、将校を雇い、キャリアポイントを振る。
6. 次の小戦闘または大戦闘に出す軍団を選ぶ。
7. 戦闘中の損耗、将校負傷、補給消費、部隊経験が次に残る。

この往復が本作の核。戦闘が単発パズルではなく、育成リソースの消費/投資判断になる。

## 主なシステム階層

| 階層 | 内容 | 育成への関係 |
| --- | --- | --- |
| 将軍/プレイヤー | Politics, Economy, Medicine, Training, Army Organization, Logistics, Reconnaissance | 軍全体の成長方針を決めるメタ育成。 |
| 軍団/師団/旅団 | Corps / Division / Brigade の階層編成 | Army Organizationと将校ランクで規模が広がる。 |
| 部隊 | Infantry, Cavalry, Artillery, Skirmishers | 戦闘経験でスキルと星が伸びる。 |
| 将校 | Brigade / Division / Corps 指揮官 | 昇進、負傷、戦死、Perkがある。 |
| 兵員 | Rookie / Veteran 補充 | 安い新人補充は能力を薄め、高いベテラン補充は能力を維持する。 |
| 武器/補給 | Armory, Shop, Supply Wagon | 武器在庫、鹵獲、補給消費が戦力を左右する。 |
| 評判 | Reputation | 士気補正と政府支援購入の通貨を兼ねる。 |
| 敵軍情報 | Intelligence Service | 敵のArmy/Training/Armoryを見て準備を変える。 |

## 戦闘の位置づけ

本作の戦闘は「勝てば終わり」ではない。

- 勝利しても損害が大きいと次が苦しい。
- ベテラン部隊を前線に出しすぎると、強いが補充費が高くなる。
- 将校が死ぬと、部隊効率が落ち、再配置や雇用が必要になる。
- 補給を消耗しすぎると、戦闘中の継続火力が落ちる。
- 敵に損害を与えると、次戦の敵戦力にも影響し得る。

つまり、戦闘は「キャンペーン育成の成果を使う場所」であり、同時に「次の育成状態を壊す場所」でもある。

## 陣営差

公式ガイドでは、南軍は新兵の質が高いが、武器・ゴールド・兵員が限られる。一方、北軍は資源が多いが新兵の質が低く、育成に時間がかかる。

この差が、同じ育成システムでもプレイ感を変える。

- 南軍: 少ない資源で精鋭を守るゲームになりやすい。
- 北軍: 多い資源で量を整え、未熟な兵を育てるゲームになりやすい。

## UI/導線

Campは `Army`, `Armory`, `Barracks`, `Career` を中心に構成される。

- `Army`: 軍団/師団/旅団の編成、部隊補充、武器変更、将校差し替え。
- `Armory`: 武器購入、在庫、鹵獲品、売却。
- `Barracks`: 将校雇用、予備将校、負傷将校。
- `Career`: キャリアポイント、評判、戦歴、メダル。
- `Battle Map`: 次戦、サイドバトル、報酬、配置枠、敵情報。

戦闘前にやることが多いが、全部が「次の戦闘で誰を生かすか」に収束している。

## 面白さの基本設計

本作の面白さは、数値育成ではなく「判断の記憶」にある。

- あの旅団を温存したから終盤で切り札になる。
- あの戦闘で無理をしたから、補充で資金が消えた。
- 高級武器を誰に渡すかで軍の性格が変わる。
- 将校が死ぬと、ただの損失ではなく指揮系統の穴になる。
- 兵力を増やすと安心だが、維持費、武器、敵スケール、指揮官不足が発生する。

育成ゲームとして見ると、これは「強化して勝つ」ではなく「勝つためにどこまで消耗を許すか」のゲーム。
